# GET /users/:id

Retrieves a specific user by ID.

```js
{
  "meta": {
    "status": 200
  },
  "data": {
    "id": 3,
    "username": "TiltifyUser",
    "slug": "tiltify-user",
    "thumbnail": {
      "src": "https://asdf.cloudfront.net/asdf.jpg",
      "alt": "synthesize distributed solutions",
      "width": 200,
      "height": 200
    },
    "status": "completed"
  }
}
```
